## Executar Compilador

Se estiver em um ambiente Windows execute o arquivo de execução do Windows:

```ps1
.\compiladorWindows.ps1
```

Se estiver em um ambiente Linux ou Unix execute o seguinte arquivo executável:

```bash
.\compiladorUnix
```
